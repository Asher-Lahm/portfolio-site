Software Projects

Computer Friends

Computer Friends is an interactive piece of art that generates friends.

Friends are randomly generated collections of patterns. Each is unique, with its own randomly generated color palette, randomly selected shapes, and randomized wave-like movements.

See if you can find a friend that resonates with you, or one that you think is able to represent yourself well!

It is currently hosted on p5.js, and can run in your browser!

mininova

mininova is an interactive music visualizer, featuring many functions to create fun and flashy visuals synced to music.

It features the central nova, which pulses to the music, as well as sound-reactive shapes, that spawn in around the nova and are pulled in by it.

Some of the user controls include changing the color palette, increasing and decreasing sound sensitivity, moving the nova around the screen, manually spawning shapes, changing the spin intensity, and typing in lyrics live to then be shown on-screen.

It is also currently hosted on p5.js, and can run in your browser!

Video Projects

Nebbys Intro

This is a motion graphics project I worked on in class. It was in the process of making this project that I really began to feel comfortable in After Effects.

I was able to incorporate a lot of different mediums, like Blender and TouchDesigner. I also picked up the expressions editor. The animated sine wave were animated entirely using expressions!

I modified my Computer Friends project to export friends as SVG files, and I then animated them here!

I became fascinated by the text effects included in After Effects. I wanted to apply these effects to my shapes from Computer Friends. After some tinkering, I created my own font comprised of shapes, which I could then animate using text effects! You can see this effect in the lines of shapes that rush across the screen.

Animated Text

Warning: Contains harsh language!

This projet was the final for my motion graphics class.

I animated a humorous scene from one of my favorite shows, It's Always Sunny in Philadelphia!

I challenged myself to learn some new effects using random noise, masks, and blending modes. I followed some tutorials online for certain effects, like the burning paper or the metallic text. I then used that knowledge to create my own animated chalkboard effect!
